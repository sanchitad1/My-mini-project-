# Tic-Tac-Toe Game (Python)

## Overview
This is a simple command-line Tic-Tac-Toe game developed in Python. Two players can play alternately by entering row and column numbers. The game automatically checks for a winner or a tie.

## Features
- Two-player gameplay
- Input validation
- Winner detection
- Tie detection
- Easy-to-read game board

## Technologies Used
- Python 3
